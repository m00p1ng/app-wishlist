# App Wishlish
> Watch app for discounting

## Installation
```
$ lein deps
```

## Configuration

**.env**
```
EMAIL={FOR SENDING}
PASS={YOUR PASSWORD}
TO={SEND TO?}
```

**app_list.json**
List all you wish
```json
[
  {
    "name": "whatever",
    "id": "appId"
  }
]
```

## RUN
```
$ lein run
```
